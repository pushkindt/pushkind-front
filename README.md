# pushkind-front

A storefront app for pushkind-orders to allow clients browse products,
categories, tags, and create orders, having authenticated with an OTP SMS to
their phone numbers.

## Environment

1. Copy `.env.example` to `.env.local`.
2. Set `VITE_HUB_ID` to the hub identifier provided by Pushkind.
3. Set `VITE_API_URL` to the base URL for the Pushkind Orders API.
